## Code Editor

### Technology
   * Python
      #### Libraries
        * Tkinter
        * Keyboard
        * subprocess
        * shutil
        * pyautogui

### Description
   * It is Desktop Application .
   * It is use to write , edit , run your code .
   * It also provide auto complete feature in html language.
   * It support python,html,js,c,c++,java.
   * It can also work as notepad.
